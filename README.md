# Link na Bio - Improvet

Página de link na bio do Instagram para a Improvet. Design responsivo com aurora animada, cascata de conteúdo e card institucional.

## Arquivos principais

- `Link na Bio.dc.html` — Design Component editável (use isso no Antigravity com Claude)
- `Link na Bio.html` — Versão standalone (tudo embutido, pronto pra usar)
- `uploads/` — Imagens e assets
- `support.js` — Runtime do Design Component

## Como editar

1. Abra `Link na Bio.dc.html` no editor
2. Faça as alterações
3. Commit e push

## Deploy

Suba pra Vercel/GitHub Pages apontando a raiz do projeto.
